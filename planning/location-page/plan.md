# Location Page Module - Planning

## Overview
Location page displaying Aeroland project location details, map, pocket information, and geographical advantages.

## Objectives
- Show project location clearly
- Display map and directions
- Explain geographical advantages
- Show proximity to key landmarks
- Information about both pockets (A & B)

## Key Features
1. **Interactive Map**
   - Google Maps integration
   - Project markers
   - Zoom capability
   - Directions link
   - Responsive design

2. **Pocket Information**
   - Pocket A details (north, near Dyalpur/Nandiali)
   - Pocket B details (south, near Matran)
   - Individual descriptions
   - Statistics for each pocket

3. **Geographical Advantages**
   - Distance to Airport
   - Distance to Chandigarh city
   - Proximity to Aerocity
   - Proximity to Expo City
   - Road connectivity

4. **Landmarks & Attractions**
   - Nearby amenities
   - Educational institutions
   - Shopping centers
   - Hospital facilities
   - Parks and recreation

5. **Accessibility**
   - Public transport options
   - Road connectivity
   - Travel time information
   - Getting there guide

6. **Visual Aids**
   - Map image (aeroland-map.jpg placeholder)
   - Location photography
   - Proximity diagrams
